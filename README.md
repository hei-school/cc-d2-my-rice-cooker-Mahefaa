# Vary-Cooker

Vary cooker is a simple cli-like app which will help you manage a real-life like rice-cooker.
"Vary" being the malagasy word for rice.

## Description:

* ### What can vary-cooker do ?
    1. turnOn
    2. turnOff
    3. addWater 1L
    4. addRice per 1 cup
    5. startCooking
    6. stopCooking
